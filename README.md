## 開発テーマ: カスタム連携機能

VoiceOS Agent Modeで、**MCP（Model Context Protocol）を使ったカスタム連携機能**を開発します。

### 機能概要

ユーザーが声だけで外部サービスやツールとカスタム連携できる機能を構築。

### 実装アイデア

- MCPサーバー経由で外部APIと連携
- 音声コマンドで連携設定を実行
- 既存のMCPサーバーを活用した拡張

## 技術スタック
