# デモ動画シナリオ（1分間）

## 設定: 月初の経営チェック

**リアルな課題**: 4月1日に月が変わった。経営者は「先月の数字」と「今の資金状況」を早く把握したいが、freeeを開いてレポートを見る時間がない。

---

## 台本（約55秒）

| 秒数 | あなたの声 | VoiceOSの返答 |
|------|-----------|--------------|
| 0-5 | *(VoiceOS画面を見せながら)* | *(接続済みの画面)* |
| 5-12 | **「今月の損益を教えて」** | 「株式会社テックスタートの2026年4月の損益概況です。売上高320万円、売上原価120万円、売上総利益200万円、販管費150万円、営業利益50万円、経常利益49万円です。」 |
| 12-20 | **「先月と比べてどう？」** | 「売上高320万円、前月280万円、プラス40万円、プラス14.3%。営業利益50万円、前月30万円、プラス20万円、プラス66.7%。経常利益49万円、前月29万円、プラス20万円、プラス69%です。」 |
| 20-30 | **「口座の残高は？」** | 「口座残高一覧です。三井住友銀行渋谷支店に450万円、住信SBIネット銀行に120万円、事業用VISAカードがマイナス35万円、現金8万円。合計残高543万円です。」 |
| 30-42 | **「未回収の売上ある？」** | 「未決済の売上取引です。株式会社ABC商事30万円、期日3月31日、3日超過。合同会社デザインワークス45万円、期日4月30日。株式会社フューチャーテック12万円、期日4月10日。合計3件、87万円です。」 |
| 42-50 | **「販管費の内訳は？」** | 「給料手当80万円、法定福利費12万円、地代家賃20万円、広告宣伝費15万円…合計150万円です。」 |
| 50-55 | *(カメラに向かって)* **「声だけで経営数字がすべて確認できます」** | |

---

## デモのポイント

1. **最初の質問で概況把握** → 経営者が一番知りたい全体像
2. **前月比で変化を確認** → 数字の良し悪しが判断できる
3. **口座残高で資金繰り確認** → 今お金があるか
4. **未回収の催促判断** → ABC商事が3日超過 → すぐ対応すべき
5. **詳細を深掘り** → 気になった費目を掘り下げる

**ストーリー**: 概況 → 比較 → 資金 → リスク → 詳細、と自然に深掘りしていく流れ
